#  Python Test Automation Project

Bu proje, **Python** ve **Pytest** kullanılarak geliştirilmiş bir **web uygulaması test otomasyonu örneğidir**.  
Amaç, temel test otomasyonu konseptlerini öğrenmek ve portfolyo için örnek bir proje sunmaktır.

---

##  Kullanılan Teknolojiler
- **Python 3.10+**
- **Pytest** (Test otomasyonu framework)
- **Selenium** (Web tarayıcı otomasyonu)
